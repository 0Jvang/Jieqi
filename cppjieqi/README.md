# CPP README

## Quick Start:

On Ubuntu:

`mkdir build && cd build && cmake .. && make`

On Windows:

`mkdir build && cd build && cmake .. -G "MinGW Makefiles" && make`

Windows的终端字符集要设为UTF-8, 否则会出现乱码

## Players.conf

第一行表示红方, 第二行表示黑方

0表示人，1, 2, ...表示电脑, eg:

1

0

表示AIBoard1执红, 人执黑。

2

1

表示AIBoard2执红,  AIBoard1执黑。
