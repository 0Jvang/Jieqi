# CPP README

## Quick Start:

On Ubuntu:

`mkdir build && cd build && cmake .. && make`

On Windows:

`mkdir build && cd build && cmake .. -G "MinGW Makefiles" && make`

Windows的终端字符集要设为UTF-8, 否则会出现乱码

0

表示人执红, 电脑执黑。
